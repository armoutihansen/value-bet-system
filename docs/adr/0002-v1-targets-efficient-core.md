# 0002 — V1 hunts the efficient, data-rich core (measurability over edge-plausibility)

The goal is to find value bets, which tempts starting in inefficient niche markets (lower divisions, exotic markets) where mispricing is likelier. But ADR-0001 makes CLV against a sharp closing line, plus calibration, the proof of edge — and both require a sharp closing line and clean data. Those exist only in liquid, efficient markets. Niche markets have softer closing lines (a weak CLV benchmark), patchier and more leakage-prone data, and thin liquidity.

**Decision.** V1 deliberately targets the efficient, data-rich core (top leagues, liquid markets). "Efficient" is reframed as "measurable": the V1 research question is whether the model produces positive CLV against the sharp close, and in which segments. Scope then expands *empirically* toward whatever segments show signal, rather than presupposing that niche = profit.

**Consequences.** Expect modest or no edge — an acceptable result under ADR-0001. Going niche to chase edge is rejected because it discards the measuring instrument that is the project. The specific league/market list is a flexible starting point; the durable decision is the *posture* — measurability-first, efficient-core-first, expand toward signal.
